# UX/UI Спецификация

## Целевая аудитория
Женщины 25-40 лет — основные участницы маркетинговых акций.

## Цветовая палитра

| Цвет | HEX | Использование |
|------|-----|---------------|
| Основной текст | #5A4A4A | Заголовки, важный текст |
| Вторичный текст | #8B7B7B | Подписи, описания |
| Акцент золотой | #D4AF7A | Кнопки, иконки, победа |
| Акцент розовый | #D4A5A5 | Проигрыш, декор |
| Лавандовый | #E8D5E8 | Фоновые градиенты |
| Персиковый | #FFE5D9 | Карточки, модалки |
| Telegram | #0088cc | Кнопка привязки TG |

## Типографика

- **Заголовки:** системный шрифт, 3xl-4xl, цвет #5A4A4A
- **Промокод:** моноширинный (font-mono), 5xl, uppercase, tracking-wide
- **Основной текст:** 14-16px, font-light

## Компоненты

### Игровое поле
- Сетка 3x3, ячейки 80-96px
- Скругление: rounded-2xl
- Фон: белый с прозрачностью (bg-white/80)
- Тень: shadow-lg
- Анимация появления: scale + opacity

### Кнопки
- Скругление: rounded-xl / rounded-2xl
- Градиент: from-[#D4A5A5] to-[#D4AF7A]
- Hover: scale 1.02, shadow-md
- Tap: scale 0.98

### Модальные окна
- Backdrop: blur-sm + bg-black/20
- Карточка: bg-white/95, rounded-3xl, border золотой
- Декоративные уголки (border-t/b/l/r)
- Анимация: spring с rotate

### Крестик (X)
- SVG с градиентом #D4A5A5 → #D4AF7A
- Анимация: pathLength 0→1, rotate

### Нолик (O)
- SVG круг с градиентом #E8D5E8 → #D4AF7A
- Анимация: pathLength 0→1

## Анимации

| Элемент | Тип | Длительность |
|---------|-----|--------------|
| Появление страницы | fade + slide up | 0.6s |
| Ход игрока | scale spring | 0.3s |
| Модалка победы | scale + rotate spring | 0.5s |
| Конфетти | fall + rotate | 2-4s |
| Sparkles | pulse scale | 1.5s infinite |
| Фоновые круги | scale + opacity | 4-5s infinite |

## Состояния

### Кнопка TG
- **Не подключён:** border золотой, текст тёмный
- **Подключён:** border и текст #0088cc, иконка галочка

### Ячейка поля
- **Пустая:** hover scale 1.05, курсор pointer
- **Занята:** disabled, без hover
- **Ход компьютера:** cursor not-allowed

## Адаптивность

- Mobile-first подход
- Ячейки: 80px (mobile) → 96px (sm+)
- Отступы: p-4
- Высота: 100dvh без скролла

## Микровзаимодействия

1. **Hover на ячейку** — лёгкое увеличение + shimmer эффект
2. **Клик** — scale down 0.95
3. **Победа** — конфетти + sparkles вокруг модалки
4. **Копирование промокода** — иконка меняется на галочку
